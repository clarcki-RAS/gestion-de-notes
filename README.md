
# 📊 Application de gestion des moyennes

## 🚀 Description
Application web fullstack permettant de gérer les notes et calculer automatiquement les moyennes des étudiants.

## 🛠️ Technologies
- Frontend : Next.js, Tailwind CSS
- Backend : Laravel (API REST)
- Base de données : PostgreSQL

## ⚙️ Fonctionnalités
- Gestion des étudiants
- Ajout/modification des notes
- Calcul automatique des moyennes
- API REST sécurisée

## 🔗 Architecture
Frontend (Next.js) → API Laravel → PostgreSQL

## ▶️ Installation

### Backend & frontend
```bash
cd backend
composer install
php artisan serve

cd e-note-front
npm install
# ou
yarn install

npm run dev
